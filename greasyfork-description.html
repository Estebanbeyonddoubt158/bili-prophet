看视频前先知道值不值得看。AI 自动分析字幕，主页封面直出评级 + 概述 + 关键点，进度条标注关键时间轴。

✨ 功能介绍

📺 主页智能预览

  • 后台自动分析主页、搜索页、排行榜、UP主空间的视频
  • 封面直接显示 评级（值得看 / 一般 / 不值得看）、概述、关键点、时间节点
  • 鼠标移开后评级徽章保留在封面左上角
  • 深度渐变 + 双层文字阴影，任何封面颜色下都清晰可读

📋 一键复制

  • 悬停封面即显示 AI 总结
  • 「复制」按钮将评级 + 概述 + 关键点一键写入剪贴板
  • 章节标签可点击，直接跳转到对应时间点

⏱️ 进度条时间轴

  • 视频详情页进度条上自动标注 AI 分析出的关键时间节点
  • 蓝色竖线 + 标签 + 箭头，始终显示
  • 点击标签直接跳转，支持 SPA 路由

💾 缓存与配置

  • IndexedDB 本地缓存，默认 7 天有效期
  • 自定义 LLM 接口（Endpoint / API Key / 模型）
  • 评级评判标准可在设置面板自定义，留空使用内置默认规则
  • 油猴菜单 → ⚙️ 设置面板

🚀 安装步骤

  1. 安装 Tampermonkey（Chrome / Edge / Firefox）
  2. 点击页面顶部「安装」按钮，或直接访问本页面安装脚本
  3. 打开任意 B站页面，点击油猴图标 → ⚙️ 设置
  4. 填入 LLM API 地址、Key 和模型名称
  5. 回到 B站主页，悬停封面即可看到 AI 评级

⚙️ 配置项说明

配置项 说明 默认值
LLM Endpoint OpenAI 兼容格式 API 地址 api.openai.com
API Key 对应服务的密钥 —
模型 推荐 gpt-4o-mini / deepseek-chat gpt-4o-mini
触发方式 悬停 or 点击触发分析 悬停 800ms
评级评判标准 自定义 rating 判断逻辑，注入提示词 内置默认规则
缓存有效期 分析结果本地缓存天数 7 天

📋 使用要求

  • Tampermonkey（Chrome / Edge / Firefox）
  • OpenAI 兼容格式的 LLM API（支持流式输出）
  • 视频需有 CC 字幕

🔒 隐私说明

所有数据仅在本地处理。字幕内容仅发送至你填写的 LLM API 地址，不经过任何第三方中转。分析结果缓存在本地 IndexedDB 中。
